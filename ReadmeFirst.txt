In order to compile this project you must add two symbolic link:

- A symbolic link named "AdiumBuild" in project root pointing to build directory of Adium (e.g. Adium/build/Release for Adium 1.4)

- A symbolic link named "InternalFrameworks" in project root pointing to Frameworks directory of Adium sources (e.g. Adium/Frameworks)

It was tested with Adium 1.3.6 sources and Adium 1.4hg sources (09/06/2009)
